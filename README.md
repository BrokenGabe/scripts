## Wallterm.sh

This script creates a terminal window pinned to the desktop background, useful for running things like htop or gotop

##### Dependencies
~~~
1 WMTCRL
~~~
To install WMCTRL
~~~
1. LinuxMint/Ubuntu/Debian
    * sudo apt-get install wmctrl -fy
2. Arch/Mangaro
    * pacman -S wmctrl
~~~
